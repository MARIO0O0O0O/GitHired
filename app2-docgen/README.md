# App 2 — Doc Generator
- What it does: Generates ATS-optimized resumes and cover letters using approved professional inventory.
- Input it needs: `master_professional_inventory.md`, Jinja2 templates, and job-specific role information.
- Output it produces: `resume_[jobname]_[date].md` and `cover_letter_[jobname]_[date].md`.
