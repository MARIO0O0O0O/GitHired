# App 4 — Match Engine
- What it does: Compares approved career inventory against job postings to produce qualification reports.
- Input it needs: `master_professional_inventory.md` and structured job postings JSON.
- Output it produces: `match_report_[jobname]_[date].md` with MEETS MINIMUM/PARTIAL MATCH/DOES NOT MEET scores.
